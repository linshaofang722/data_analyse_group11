# 35个主要城市财政与房地产投资依赖度演变分析项目

本项目旨在通过量化数据分析，深度剖析中国35个主要城市在过去近二十年（2006-2024年）间，地方财政健康状况与房地产投资依赖度之间的演变规律与空间结构差异。本文档详细说明了项目的研究目的、整体架构、目录结构、开发环境以及核心数据字段的映射关系。

---

## 1. 研究目的

本研究的核心目标是探究“房地产周期波动”与“地方财政承压”之间的内在联系。通过构建“财政缺口率”与“房地产投资依赖度”两大核心指标，项目试图解答以下问题：
- **时间维度**：在稳步增长期、棚改调控期与深度调整期三大宏观周期中，全国主要城市的财政与地产依赖度呈现何种演变趋势？
- **空间维度**：在当前的深度调整期（2021-2024年），不同区域、不同能级的城市在抗风险能力上出现了怎样的结构性分化？
- **政策导向**：通过象限聚类分析，识别出深陷“转型阵痛”与成功迈入“健康发展”的典型城市，为地方债务化解与产业转型提供实证依据。

---

## 2. 整体思路与处理阶段

本项目采用“自上而下”的结构化数据分析方法，整体架构分为以下五个递进阶段：

### 第一阶段：数据清洗与预处理
- 针对多源面板数据进行合并与对齐。
- 处理缺失值与异常值，统一城市名称与年份格式，构建标准化的时序面板数据集。

### 第二阶段：核心指标工程
- 基于基础宏观经济数据，衍生出本研究的两大核心分析变量：
  1. 财政缺口率 = (地方一般公共预算支出 - 地方一般公共预算收入) / 地区生产总值
  2. 房地产投资依赖度 = 房地产开发投资额 / 地区生产总值

### 第三阶段：时序特征分析 (宏观全景与微观切片)
- 计算35个城市的年度均值，绘制全国大盘的宏观全景趋势图。
- 选取东、中、西部共9个典型代表城市，绘制“城市个体 vs 全国均值”的同框双Y轴对比图，揭示个体与大盘的共振与背离。

### 第四阶段：空间维度象限聚类分析
- 截取“深度调整期”（2021-2024年）的数据计算均值。
- 以35城的中位数为十字交叉基准线，构建二维散点图，将城市划分为四大生态位：转型阵痛区、内生乏力区、健康发展区、传统红利区。

### 第五阶段：核心结论提炼
- 整合图表与数据输出，提炼宏观趋势与空间格局的规律，形成最终的研究洞察与报告框架。

## 5. 数据字典与字段说明

本项目涉及的数据字段分为“原始基础指标”与“衍生核心指标”两类。具体映射关系如下表所示：

### 5.1 原始基础指标 (源文件字段)

| 字段名称 | 指标含义 | 单位 |
| :--- | :--- | :--- |
| `city` | 城市名称 | - |
| `year` | 统计年份 | 年 |
| `city_expenditure` | 地方一般公共预算支出 | 亿元 |
| `city_income` | 地方一般公共预算收入 | 亿元 |
| `gdp` | 地区生产总值（当年价格） | 亿元 |
| `individual_deposit` | 住户存款余额 | 亿元 |
| `re_avg_price` | 商品房平均销售价格 | 元/平方米 |
| `re_completion` | 房地产开发企业竣工房屋面积 | 万平方米 |
| `re_construction` | 房地产开发企业施工房屋面积 | 万平方米 |
| `re_dev_inv` | 房地产开发投资额 | 亿元 |
| `re_land_purchase_area` | 房地产开发企业购置土地面积 | 万平方米 |
| `re_sales_area` | 商品房销售面积 | 万平方米 |

### 5.2 衍生核心指标 (生成表格字段)

| 字段名称 | 指标含义 | 计算公式与业务逻辑 |
| :--- | :--- | :--- |
| `gap_to_gdp` | 财政缺口率 | `(city_expenditure - city_income) / gdp`。衡量地方财政收支失衡程度，数值越高代表财政压力越大。 |
| `re_invest_dependence` | 房地产投资依赖度 | `re_dev_inv / gdp`。衡量地方经济增长对房地产投资的依赖程度，数值越高代表经济结构越传统。 |
| `quadrant` | 所属象限分类 | 基于2021-2024年均值与整体中位数的比较结果，分为：转型阵痛区、内生乏力区、健康发展区、传统红利区。 |
